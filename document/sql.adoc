[[la_commande_select]]
== La commande SELECT

La commande `SELECT` permet de choisir la ou les *colonnes* que l'on souhaite récupérer.

[source%linenums,sql]
----
SELECT nom FROM client;
SELECT nom, prenom FROM client;
----

Il est possible de faire des calculs dans une `SELECT` :

[source%linenums,sql]
----
SELECT 1+3
SELECT nom, prenom, 5*2 FROM client;
----
